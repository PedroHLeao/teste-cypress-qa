# Projeto de Automação - Cypress + Cucumber

## 📌 Descrição
Projeto de automação de testes web e API utilizando Cypress, Cucumber (BDD) e JavaScript.
As credenciais fornecidas no desafio não estavam válidas, então foi criado um usuário para execução dos testes.

---

## 🚀 Tecnologias utilizadas
- Cypress
- Cucumber
- JavaScript
